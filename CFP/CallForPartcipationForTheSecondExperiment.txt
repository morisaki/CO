Call for participation

The results of your code comprehension will be published in an anonymized format through papers, blog posts, conference presentations, media outlets, and articles. This study is positioned as a code comprehension experiment.Participants will comprehend approximately 400 lines of Java source code prepared by the organizers using generative AI assistance,and then answer 9 questions and complete a questionnaire about the source code. The estimated time required is approximately one and a half hours.

Assumed Skills of Participants
Participants are assumed to possess the following skills: a basic understanding of the Java language specification and experience in Java development; a basic understanding of networking concepts such as sending and receiving data (no knowledge of specific class libraries is assumed); access to a generative AI tool and some experience using it for code comprehension. Participants who have not used generative AI before are welcome as 
long as they understand how to use it (e.g., how to write prompts). Any type of generative AI tool is acceptable.

Use of Submitted Results 
All results will be anonymized and published in the form of statistical summaries through papers, blog posts, conference presentations, media outlets, and 
articles.

Steps Required for Participation
1. Register as a participant: The questionnaire on this page will ask about your knowledge of design patterns, contact information, and development experience. (For research purposes, participants will be divided into groups, and this information will be used for group assignment.)
2. Comprehend approximately 400 lines of Java source code using generative AI: After the source code is sent to you, please comprehend it independently by following the provided instructions.
3. Submit the results of your source code comprehension: Please submit your answers to the questions and the questionnaire about the source code.

Important Notes
1. Please ensure that you have sufficient time available for the comprehension task (up to approximately two hours).
2. Please prepare your own generative AI tool for use during the code comprehension task. If you use a free version, please be aware of any usage limitations.
3. The results of the Java source code comprehension task will be anonymized and published through papers, blog posts, conference presentations, media outlets, and articles. Anonymized data may also be disclosed or provided to third parties.
4. You will be asked to share the content of the prompts you submitted to generative AI during the code comprehension task. The prompt content may also be anonymized and published through papers, blog posts, conference presentations, and similar outlets, as described in Note 3.
5. The copyright of the target Java source code is held by other researchers and may not be reproduced or used for secondary purposes.
6. Participants who have already completed the code comprehension task at another session are not required to participate again
